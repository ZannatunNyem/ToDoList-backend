# 📝 ToDo List Backend

## 📖 Overview

The ToDo List Backend is a RESTful API built with Node.js, Express.js, and MongoDB. It provides CRUD operations for managing tasks and serves as the backend for the ToDo List frontend application.

---

## ⚙️ Features

✔ Create new tasks

✔ Update existing tasks

✔ Delete tasks

✔ MongoDB database integration

✔ Error handling

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| Node.js | Runtime Environment |
| Express.js | Backend Framework |
| MongoDB | Database |
| dotenv | Environment Variables |

---

## 📡 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/todo-list-backend.git
```


### Install Packages

```bash
npm install
```


### Start Development Server

```bash
npm run dev
```
